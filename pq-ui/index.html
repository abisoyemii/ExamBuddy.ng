About UI Post-UTME Screening

📋

Screening Format

UI conducts an online Post-UTME screening test. It covers your JAMB subjects. The exam is Computer Based, 50–60 questions, 30–45 minutes. Questions test understanding, not memory.

📊

Admission Aggregate

UI combines your JAMB score and Post-UTME score into an aggregate. O'Level grades (WAEC/NECO) also contribute. All three components count towards final ranking.

🎯

What UI Tests

UI Post-UTME questions are known to be more analytical and reasoning-based than JAMB. Expect inference questions, data interpretation, and application of concepts — not just recall.
