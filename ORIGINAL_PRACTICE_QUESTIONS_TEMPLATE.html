🔢 JAMB Mathematics - Practice Set 1

Q1

Algebra - Simultaneous Equations

Question: Solve the simultaneous equations:

2x + 3y = 12
x - y = 1

Correct Answer: A

Solution:

From x - y = 1, we get x = y + 1

Substitute into 2x + 3y = 12:

2(y + 1) + 3y = 12

2y + 2 + 3y = 12

5y = 10

y = 2, therefore x = 3

Q2

Geometry - Angles

Question: In a triangle ABC, angle A = 45°, angle B = 60°. Find angle C.

Correct Answer: B

Solution: Sum of angles in a triangle = 180°

A + B + C = 180°

45° + 60° + C = 180°

C = 180° - 105° = 75°

📚 WAEC Literature in English - Practice Set 1

Q1

Poetry Analysis

"The golden sun descends beyond the hill,
Painting the sky in hues of red and gold.
The evening breeze blows gentle, soft and still,
As nature's beauty slowly starts to unfold."

Question: What literary device is primarily used in these lines?

Correct Answer: B

Explanation: Imagery is the use of vivid descriptive language to create mental images. The poem uses visual descriptions (golden sun, red and gold hues, evening breeze) to paint a picture.
